# CodeHubot PBL (Project-Based Learning)

CodeHubot 平台的项目式学习子系统。旨在提供以项目为核心的学习环境，让学生通过组队协作完成具有挑战性的项目，培养解决实际问题的能力。


## 🛠 技术栈

- **前端**: Vue.js 3 + Vite
- **后端**: Python FastAPI
- **数据库**: MySQL (共享主系统核心表)

## 🚀 快速开始

### Docker 启动

```bash
docker-compose up -d
```

### 目录结构

- `frontend/`: 前端项目源码
- `backend/`: 后端 API 服务源码
- `SQL/`: 数据库脚本
